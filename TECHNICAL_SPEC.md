## 2. Концепция игры

### 2.1 Описание
Neon Breakout — современная интерпретация классической игры Breakout/Arkanoid с неоновой киберпанк-эстетикой. Игрок управляет платформой внизу экрана, отбивая мяч, который разбивает блоки в верхней части игрового поля.

### 2.2 Уникальные особенности
- Неоновый визуальный стиль с glow-эффектами
- Система комбо-множителей
- Бонусы (power-ups), выпадающие из блоков
- Несколько типов блоков с разной прочностью
- Адаптивный дизайн (desktop + mobile)
- Звуковые эффекты через Web Audio API

## 3. Функциональные требования

### 3.1 Игровая механика

| Элемент | Описание |
|---------|----------|
| Платформа | Горизонтальная панель, управляемая игроком |
| Мяч | Отскакивает от стен, платформы и блоков |
| Блоки | Разрушаются при попадании мяча |
| Жизни | 3 жизни, теряется при падении мяча |
| Счёт | Начисляется за разбитые блоки |

### 3.2 Типы блоков

| Тип | Цвет | Прочность | Очки |
|-----|------|-----------|------|
| Обычный | Голубой | 1 удар | 10 |
| Усиленный | Оранжевый | 2 удара | 25 |
| Прочный | Красный | 3 удара | 50 |

### 3.3 Power-ups (бонусы)

| Бонус | Эффект | Длительность |
|-------|--------|--------------|
| Широкая платформа | Увеличивает платформу на 50% | 10 сек |
| Мультибол | Добавляет 2 дополнительных мяча | До потери |
| Медленный мяч | Замедляет мяч на 30% | 8 сек |

### 3.4 Управление

**Desktop:**
- Стрелки влево/вправо или A/D — движение платформы
- Пробел — запуск мяча / пауза

**Mobile:**
- Касание и движение пальцем — управление платформой
- Тап — запуск мяча
